Создайте папку test в корневом каталоге сервера и загрузите туда файлы из репозитория.
В файле config/db.php укажите пароль для подключения к базе данных.
С помощью yii2_test.sql файла импортируйте базу данных MySQL (yii2_test).
Через консоль в папке test запустите команду "composer update".
Для запуска зайдите на http://имя_сервера/test/web.
